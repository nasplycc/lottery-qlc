# 七乐彩选号系统

[![GitHub](https://img.shields.io/github/v/tag/nasplycc/lottery-qlc?label=version)](https://github.com/nasplycc/lottery-qlc)
[![License](https://img.shields.io/github/license/nasplycc/lottery-qlc)](LICENSE)

中国福利彩票七乐彩（QLC）智能选号与复盘系统，基于历史数据的概率分析生成选号推荐。

## 📊 系统特点

- **历史数据库**：已收录 2007-2026 年共 2921 期开奖数据
- **概率分析**：热号/冷号统计、频率分析、遗漏值计算
- **规则筛选**：和值、奇偶比、分区比、连号约束
- **自动生成**：单式选号 + 复式选号
- **开奖复盘**：命中对比、奖金计算、规则优化建议

## 🎯 开奖信息

| 项目 | 详情 |
|------|------|
| **彩票类型** | 中国福利彩票 - 七乐彩 |
| **开奖日期** | 每周一、三、五 |
| **开奖时间** | 21:15 |
| **玩法规则** | 从 01-30 选 7 个红球 + 从 01-30 选 1 个特别号 |
| **单注成本** | 2 元 |
| **奖级数量** | 7 个奖级 |

## 🚀 快速开始

### 环境要求

- Python 3.8+
- 依赖库：`requests`, `beautifulsoup4`

```bash
pip install requests beautifulsoup4
```

### 生成选号

```bash
cd lottery-qlc
python3 scripts/generate_qlc.py
```

**输出示例**：
```

  推荐1: 01 05 06 12 16 24 25 + 特 02
```

### 开奖复盘

```bash
python3 scripts/review_qlc.py
```

功能：
- 自动获取最新开奖结果
- 对比上期选号命中情况
- 计算奖金总额
- 提供规则优化建议

### 更新历史数据

```bash
python3 scripts/update_qlc_history.py
```

自动从 500 彩票网抓取最新开奖数据，更新本地数据库。

## 📈 选号策略

核心规则（详见 `docs/strategy.md`）：

1. **和值约束**：80-150（理论和值约 108）
2. **奇偶比**：优先 3:4、4:3、5:2、2:5
3. **分区比**：三区均衡（2:2:3、2:3:2、3:2:2 等）
4. **连号处理**：允许 0-2 组连号
5. **热冷搭配**：热号 3-4 个 + 冷号 1-2 个 + 温号补足
6. **特别号**：优先选择近期热号

### 策略有效性

基于 2921 期历史数据统计：
- 和值 80-150 区间覆盖率：约 85%
- 奇偶比 3:4/4:3 覆盖率：约 60%
- 包含 1-2 组连号：约 65%

## 🔧 配置说明

编辑 `config.json` 可调整选号参数：

```json
{
  "selection_rules": {
    "min_sum": 80,
    "max_sum": 150,
    "odd_even_ratios": ["3:4", "4:3", "5:2", "2:5"],
    "zone_ratios": ["2:2:3", "2:3:2", "3:2:2"],
    "consecutive_max": 2
  }
}
```
