# 七乐彩选号系统

中国福利彩票七乐彩（QLC）智能选号与复盘系统。

## 开奖信息

- **开奖日期**：每周一、三、五
- **开奖时间**：21:15
- **玩法规则**：从 01-30 选 7 个红球 + 从 01-16 选 1 个特别号

## 目录结构

```
lottery-qlc/
├── config.json              # 配置参数
├── scripts/
│   ├── generate_qlc.py      # 选号生成器
│   ├── review_qlc.py        # 开奖复盘
│   └── update_qlc_history.py # 历史数据更新
├── data/
│   └── qlc_history.csv      # 历史开奖数据
├── outputs/                  # 选号输出
└── docs/
    └── strategy.md          # 策略说明
```

## 快速使用

### 生成选号

```bash
cd /home/Jaben/.openclaw/workspace-finnace-bot/lottery-qlc
python3 scripts/generate_qlc.py
```

输出：
- 5 注单式推荐（每注 2 元）
- 1 注 8+1 复式推荐（16 元）

### 开奖复盘

```bash
python3 scripts/review_qlc.py
```

对比最新开奖结果与上期选号，输出命中情况和奖金。

### 更新历史数据

```bash
python3 scripts/update_qlc_history.py
```

⚠️ 当前数据接口待完善，需手动导入历史数据。

## 中奖规则

| 奖级 | 条件 | 奖金 |
|------|------|------|
| 一等奖 | 7 红 +0 特 | 浮动（约 100 万） |
| 二等奖 | 7 红 +1 特 | 浮动（约 10 万） |
| 三等奖 | 6 红 +1 特 | 10,000 元 |
| 四等奖 | 6 红 +0 特 或 5 红 +1 特 | 200 元 |
| 五等奖 | 5 红 +0 特 或 4 红 +1 特 | 50 元 |
| 六等奖 | 4 红 +0 特 或 3 红 +1 特 | 10 元 |
| 七等奖 | 3 红 +0 特 或 2 红 +1 特 | 5 元 |
